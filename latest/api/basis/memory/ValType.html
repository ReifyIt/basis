basis.memory

ValType

abstract class ValType[T] extends DataType[T]

A by-value data storage strategy. Value types define an isomorphism between fixed-length byte sequences and class instances.

Frames

A value type's alignment, offset and size constitute its frame; the alignment must evenly divide the offset and size. Value types must never access more than size - 1 bytes beyond a given address. Assume proper alignment of provided addresses.

T

the modeled instance type.
