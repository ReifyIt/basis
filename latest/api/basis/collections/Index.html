basis.collections

Index

trait Index[+A] extends Equals with Family[Index[_]] with Seq[A]

An indexed sequence. Indexes can randomly access their elements.

Extensions

Add operations to this interface by importing one of these use cases:

  • import basis.sequential.general._
    • adds reductive operations (foreach, fold, reduce, etc.).
  • import basis.sequential.strict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds eager transformations (map, flatMap, filter, etc.).
  • import basis.sequential.nonstrict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds lazy transformations (map, flatMap, filter, etc.).

The following classes implement the extensions to this interface:

  • GeneralIndexOps implements reductive operations (foreach, fold, reduce, etc.).
  • StrictIndexOps implements eager transformations (map, flatMap, filter, etc.).
  • NonStrictIndexOps implements lazy transformations (map, flatMap, filter, etc.).
