basis.collections

Collection

trait Collection[+A] extends Family[Collection[A]] with Enumerator[A]

A traversable collection. Collection declares a protected foreach method that traverses its elements. To traverse a collection, invoke basis.collections.traverse, or add a public foreach implementation by importing basis.sequential.general._.

Extensions

Add operations to this interface by importing one of these use cases:

  • import basis.sequential.general._
    • adds reductive operations (foreach, fold, reduce, etc.).
  • import basis.sequential.strict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds eager transformations (map, flatMap, filter, etc.).
  • import basis.sequential.nonstrict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds lazy transformations (map, flatMap, filter, etc.).

The following classes implement the extensions to this interface:

  • GeneralCollectionOps implements reductive operations (foreach, fold, reduce, etc.).
  • StrictCollectionOps implements eager transformations (map, flatMap, filter, etc.).
  • NonStrictCollectionOps implements lazy transformations (map, flatMap, filter, etc.).
