basis.collections

Container

trait Container[+A] extends Family[Container[_]] with Collection[A]

An iterable collection. Container declares an iterator method that returns a new Iterator over its elements.

Extensions

Add operations to this interface by importing one of these use cases:

  • import basis.sequential.general._
    • adds reductive operations (foreach, fold, reduce, etc.).
  • import basis.sequential.strict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds eager transformations (map, flatMap, filter, etc.).
  • import basis.sequential.nonstrict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds lazy transformations (map, flatMap, filter, etc.).

The following classes implement the extensions to this interface:

  • GeneralContainerOps implements reductive operations (foreach, fold, reduce, etc.).
  • StrictContainerOps implements eager transformations (map, flatMap, filter, etc.).
  • NonStrictContainerOps implements lazy transformations (map, flatMap, filter, etc.).
