basis.collections

Stack

trait Stack[+A] extends Equals with Family[Stack[_]] with Side[A]

A last-in first-out sequence. The right-associative :: operator puts an element at the head of a stack.

Extensions

Add operations to this interface by importing one of these use cases:

  • import basis.sequential.general._
    • adds reductive operations (foreach, fold, reduce, etc.).
  • import basis.sequential.strict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds eager transformations (map, flatMap, filter, etc.).
  • import basis.sequential.nonstrict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds lazy transformations (map, flatMap, filter, etc.).

The following classes implement the extensions to this interface:

  • GeneralSideOps implements reductive operations (foreach, fold, reduce, etc.).
  • StrictSideOps implements eager transformations (map, flatMap, filter, etc.).
  • NonStrictSideOps implements lazy transformations (map, flatMap, filter, etc.).
