Deque

trait Deque[+A] extends Family[Deque[_]] with Stack[A]

A double-ended queue. Deques efficiently decompose into a head element and a tail deque, or an init deque and a last element.

Extensions

Add operations to this interface by importing one of these use cases:

  • import basis.sequential.general._
    • adds reductive operations (foreach, fold, reduce, etc.).
  • import basis.sequential.strict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds eager transformations (map, flatMap, filter, etc.).
  • import basis.sequential.nonstrict._
    • adds reductive operations (foreach, fold, reduce, etc.).
    • adds lazy transformations (map, flatMap, filter, etc.).

The following classes implement the extensions to this interface:

  • GeneralDequeOps implements reductive operations (foreach, fold, reduce, etc.).
  • StrictDequeOps implements eager transformations (map, flatMap, filter, etc.).
  • NonStrictDequeOps implements lazy transformations (map, flatMap, filter, etc.).
